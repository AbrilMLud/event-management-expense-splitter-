# Event Management & Expense Splitter 📊💰

A clean and efficient Python script designed to streamline event organization and automate financial balances. 

### Key Features:
* **Event Management:** Easily organize participants and logistics.
* **Expense Splitting:** Automatically calculates who owes what, ensuring fair and accurate cost distribution.
* **Smart Balancing:** Reduces transaction chaos by optimizing peer-to-peer paybacks.
